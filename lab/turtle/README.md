# Drawing a polygon

Draws a turtle and it can be moved with w,a,s,d; to scale use + and -, to rotate i,j,k,l

## Requirements

* OpenGL
* c++

## Running

```bash
g++ main.cpp -lglut -lGL -lGLEW -lGLU
```
